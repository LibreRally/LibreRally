# LibreRally Docs

LibreRally is an open-source rally racing simulation built with Stride and .NET 10.

This site is published with [DocFX](https://dotnet.github.io/docfx/) and includes:

- project documentation for gameplay systems and content
- generated API reference from the `LibreRally` game library XML comments

## Getting started

- Explore the [reference vehicle documentation](vehicles/basic_car.md)
- Browse the generated [API reference](api/index.md)

## Projects covered by this site

- `LibreRally` for core gameplay systems
- `docs/` for conceptual documentation and guides
